수자원공사 공공데이터 활용 방안 공모전에 참여하게되었습니다.

개발 기간은 한달남짓한 기간으로 AI 시계열 수위 예측 모델을 활용한 안드로이드 네이티브 어플리케션을 만들 예정입니다.

# Model
시계열 예측 모델중 하나인 **ARIMA 모델**을 사용해 과거 공공데이터를 기반으로 미래 지하 수위를 예측합니다.

이 때 자료는 일자료를 사용해 예측을 진행할 예정이기에 제 개인 Macbook 에서 local로 모델을 돌려 output image 를 생성하고 이미지를 ec2 server(t2.micro) 에 업로드하여 구축할 예정입니다.

# Backend
백엔드 구성은 간단히 이미지 파일을 전송하는 역할만 수행하므로 python flask 를 사용하여 구성하고 웹서버는 nginx 를 통해 배포할 예정입니다.

이때 ec2 t2.micro 서버 ram 이 100mb 로 만약 배포에 어려움이 있을시 다른 서버를 찾아봐야하므로 빠르게 backend 구축후 테스트 예정입니다.

# Android App
안드로이드 앱은 android native(kotlin) 을 통해 개발할 예정이며, 네이버 지도 API 를 통해 제주도 지역 지도 기반 마커를 통해 상세 페이지 접근으로 페이지를 구성할 예정입니다.

# 진행 순서
가장 급한 모델은 현재 완료되어 있습니다.

다음으로 안드로이드 앱 배포가 빠르게 되어야할 것 같습니다.

이전 안드로이드 앱 초기 배포시 5일정도 소요되었으며 업데이트는 1~2일이면 가능하였기에,

컴팩트하게 안드로이드 앱 배포를 먼저 진행해 두고 업데이트를 통해 기능을 붙히도록 하겠습니다.

배포를 진행 한 뒤에는 바로 backend API 테스트를 진행하겠습니다.
