## NCP(Naver Cloud Platform) 서버 중단 / 반납
NCP 서버를 대여하셨다면 요금이 부과되게 됩니다. (무료 버전의 경우에도 1년 후부터는 요금 부과)
과금이 되지 않도록 중단 / 반납 하는 방법을 안내드리겠습니다.
우선 NCP 에 접속하여 내 서버 상세페이지에 접속을 합니다. 

![](/images/bae1ca9a-74b5-4fc3-8ca6-b9d4938361ff-image.png)


1. 정지 버튼을 눌러 정지를 진행하고

2. 반납(반납하지 않으면 비용이 부과됩니다.)버튼으로 반납을 진행하겠습니다.



![](/images/5c6d0c64-6e08-4429-87ec-8a44fd4563a0-image.png)



반납버튼을 누르니 친절하게 공인 IP 까지 함께 반납할지 물어보네요.

공인 IP 의 경우에도 비용(월 4000원 가량)이 부과되니 필요없으시다면 같이 반납을 진행해주세요. (번거롭게 공인IP 반납 후 서버 반납을 할 필요가 없습니다.)


![](/images/ffcfc4aa-5ebe-43d6-9936-4bd437a18431-image.png)



반납이 완료되면 초기 서버 구성 화면이 나타나게됩니다.


## 서버 설정
ncp 서버 생성 과정은 이전에도 설명했으니 넘어가도록 하겠습니다.

저는 이번에는 cpu2-ram:8gb 서버로 구성하여 동일하게 서버 설정을 진행하도록 하겠습니다.

이번 글은 ncp 서버 중단/반납에 대한 글이므로 다음 글부터 새로운 서버에서 nginx-flask 를 통한 CI/CD 구축, ssl 인증서 발급을 통한 https 설정, frontend 서버 배포등을 진행해 보도록하겠습니다.
